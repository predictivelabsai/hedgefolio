DATABASE SCHEMA
================================================================================

sec_event table:
  - event_id (PRIMARY KEY)
  - accession_number (UNIQUE)
  - filing_date (indexed)
  - form_type (indexed) - '13D' or '13G'
  - cik
  - filer_name (indexed)
  - target_company_name (indexed)
  - target_ticker (indexed)
  - stake_percentage
  - shares_owned
  - shares_outstanding
  - filing_status
  - amendment_number
  - is_amendment

================================================================================
DEPLOYMENT INSTRUCTIONS
================================================================================

1. Create database schema:
   psql "$DB_URL" -f sql/schema_events.sql

2. Install dependencies:
   pip install -r requirements.txt

3. Configure environment:
   - Set DB_URL, DB_SCHEMA, API keys in .env

4. Run Streamlit app:
   streamlit run Home.py

5. Access Events page:
   - Navigate to http://localhost:8501
   - Click "4️⃣ SEC Events" in sidebar
